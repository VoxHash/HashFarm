# Support

## Before you ask

1. Read the [README](README.md) and [monitor README](monitor/README.md).
2. Copy `scripts/common/env.template` to `.env` and configure wallet, LAN IP, and API URLs.
3. Run [`scripts/common/verify-stack.sh`](scripts/common/verify-stack.sh) on the gaming PC.
